# hash-cmp
![Rust](https://img.shields.io/badge/rust-%23000000.svg?style=for-the-badge&logo=rust&logoColor=white)\
Calculate and compare hash values quickly and easily

## Supported Hashing Algorithms
- Sha256

## How to use
```bash
./hash-cmp <file path> <hash>
```
